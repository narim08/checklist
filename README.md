2025.01.13
v2

날짜 페이지: 산책 경로 저장 기능 추가
카카오 지도 api 활용, 사용자 현위치 기반으로 지도가 나타난다. 클릭시 마커로 위도, 경도 정보가 서버에 저장된다. 지도 위에 <경로 보기> 버튼을 클릭하면 마커끼리 선으로 이어져서 표시된다. 우클릭으로 마커를 각각 지울 수 있으며, <경로 지우기> 버튼을 통해 모든 마커를 삭제할 수 있다.(모바일 지원)
